Weather Tracking System

Project Overview

I led the documentation process for this weather tracking system as part of a university project. The system provides real-time weather data to help farmers optimize irrigation, monitor soil conditions, and improve crop management. It includes tools for soil moisture tracking and customizable weather alerts, enhancing productivity and sustainability in farming.
